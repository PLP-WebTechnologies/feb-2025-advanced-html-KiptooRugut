# Advanced HTML5 & Bootstrap Demo

A demonstration of modern web development techniques using HTML5, CSS, and Bootstrap 5.

## Features

### HTML5 Semantic Elements
  - Ordered lists with Roman numerals
  - Accessible forms with validation
  - Structured tables with hover effects
  - Embedded external images

### Bootstrap Integration
  - Responsive grid system
  - Modern card components
  - Form styling and validation
  - Interactive table design

### Custom Enhancements
  - Gradient hero section
  - Animated hover effects
  - Custom form validation styling
  - Enhanced button interactions

### Prerequisites

- Modern web browser (Chrome, Firefox, Edge)
- Internet connection (for CDN resources)
